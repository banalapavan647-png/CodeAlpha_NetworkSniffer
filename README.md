# Network Sniffer

## Project Overview
This project was developed as part of the CodeAlpha Cyber Security Internship. The Network Sniffer is a Python-based tool designed to capture and analyze network packets traveling through a network interface. It helps in understanding network communication and basic packet analysis.

## Objectives
- Capture network packets in real time
- Analyze packet information
- Understand network protocols
- Learn packet sniffing techniques
- Improve cybersecurity and network monitoring skills

## Features
- Real-time packet capture
- Displays source and destination IP addresses
- Protocol identification
- Network traffic monitoring
- Basic packet analysis

## Technologies Used
- Python
- Scapy Library
- GitHub

## How to Run
1. Install Python on your system.
2. Install Scapy:
   pip install scapy
## Run the script
python network_sniffer.py

-Understanding network traffic flow
-Packet capture and inspection
-Network protocol analysis
